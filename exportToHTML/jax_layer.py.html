17   @keras_export("keras.layers.JaxLayer")
18   class JaxLayer(Layer):
19       """Keras Layer that wraps a JAX model. 
20    
21       This layer enables the use of JAX components within Keras when using JAX as 
22       the backend for Keras. 
23    
24       ## Model function 
25    
26       This layer accepts JAX models in the form of a function, `call_fn`, which 
27       must take the following arguments with these exact names: 
28    
29       - `params`: trainable parameters of the model. 
30       - `state` (*optional*): non-trainable state of the model. Can be omitted if 
31           the model has no non-trainable state. 
32       - `rng` (*optional*): a `jax.random.PRNGKey` instance. Can be omitted if the 
33           model does not need RNGs, neither during training nor during inference. 
34       - `inputs`: inputs to the model, a JAX array or a `PyTree` of arrays. 
35       - `training` (*optional*): an argument specifying if we're in training mode 
36           or inference mode, `True` is passed in training mode. Can be omitted if 
37           the model behaves the same in training mode and inference mode. 
38    
39       The `inputs` argument is mandatory. Inputs to the model must be provided via 
40       a single argument. If the JAX model takes multiple inputs as separate 
41       arguments, they must be combined into a single structure, for instance in a 
42       `tuple` or a `dict`. 
43    
44       ## Model weights initialization 
45    
46       The initialization of the `params` and `state` of the model can be handled 
47       by this layer, in which case the `init_fn` argument must be provided. This 
48       allows the model to be initialized dynamically with the right shape. 
49       Alternatively, and if the shape is known, the `params` argument and 
50       optionally the `state` argument can be used to create an already initialized 
51       model. 
52    
53       The `init_fn` function, if provided, must take the following arguments with 
54       these exact names: 
55    
56       - `rng`: a `jax.random.PRNGKey` instance. 
57       - `inputs`: a JAX array or a `PyTree` of arrays with placeholder values to 
58           provide the shape of the inputs. 
59       - `training` (*optional*): an argument specifying if we're in training mode 
60           or inference mode. `True` is always passed to `init_fn`. Can be omitted 
61           regardless of whether `call_fn` has a `training` argument. 
62    
63       ## Models with non-trainable state 
64    
65       For JAX models that have non-trainable state: 
66    
67       - `call_fn` must have a `state` argument 
68       - `call_fn` must return a `tuple` containing the outputs of the model and 
69           the new non-trainable state of the model 
70       - `init_fn` must return a `tuple` containing the initial trainable params of 
71           the model and the initial non-trainable state of the model. 
72    
73       This code shows a possible combination of `call_fn` and `init_fn` signatures 
74       for a model with non-trainable state. In this example, the model has a 
75       `training` argument and an `rng` argument in `call_fn`. 
76    
77       ```python 
78       def stateful_call(params, state, rng, inputs, training): 
79           outputs = ... 
80           new_state = ... 
81           return outputs, new_state 
82    
83       def stateful_init(rng, inputs): 
84           initial_params = ... 
85           initial_state = ... 
86           return initial_params, initial_state 
87       ``` 
88    
89       ## Models without non-trainable state 
90    
91       For JAX models with no non-trainable state: 
92    
93       - `call_fn` must not have a `state` argument 
94       - `call_fn` must return only the outputs of the model 
95       - `init_fn` must return only the initial trainable params of the model. 
96    
97       This code shows a possible combination of `call_fn` and `init_fn` signatures 
98       for a model without non-trainable state. In this example, the model does not 
99       have a `training` argument and does not have an `rng` argument in `call_fn`. 
100   
101      ```python 
102      def stateless_call(params, inputs): 
103          outputs = ... 
104          return outputs 
105   
106      def stateless_init(rng, inputs): 
107          initial_params = ... 
108          return initial_params 
109      ``` 
110   
111      ## Conforming to the required signature 
112   
113      If a model has a different signature than the one required by `JaxLayer`, 
114      one can easily write a wrapper method to adapt the arguments. This example 
115      shows a model that has multiple inputs as separate arguments, expects 
116      multiple RNGs in a `dict`, and has a `deterministic` argument with the 
117      opposite meaning of `training`. To conform, the inputs are combined in a 
118      single structure using a `tuple`, the RNG is split and used the populate the 
119      expected `dict`, and the Boolean flag is negated: 
120   
121      ```python 
122      def my_model_fn(params, rngs, input1, input2, deterministic): 
123          ... 
124          if not deterministic: 
125              dropout_rng = rngs["dropout"] 
126              keep = jax.random.bernoulli(dropout_rng, dropout_rate, x.shape) 
127              x = jax.numpy.where(keep, x / dropout_rate, 0) 
128              ... 
129          ... 
130          return outputs 
131   
132      def my_model_wrapper_fn(params, rng, inputs, training): 
133          input1, input2 = inputs 
134          rng1, rng2 = jax.random.split(rng) 
135          rngs = {"dropout": rng1, "preprocessing": rng2} 
136          deterministic = not training 
137          return my_model_fn(params, rngs, input1, input2, deterministic) 
138   
139      keras_layer = JaxLayer(my_model_wrapper_fn, params=initial_params) 
140      ```

142      ## Usage with Haiku modules 
143   
144      `JaxLayer` enables the use of [Haiku](https://dm-haiku.readthedocs.io) 
145      components in the form of 
146      [`haiku.Module`](https://dm-haiku.readthedocs.io/en/latest/api.html#module). 
147      This is achieved by transforming the module per the Haiku pattern and then 
148      passing `module.apply` in the `call_fn` parameter and `module.init` in the 
149      `init_fn` parameter if needed. 
150   
151      If the model has non-trainable state, it should be transformed with 
152      [`haiku.transform_with_state`]( 
153        https://dm-haiku.readthedocs.io/en/latest/api.html#haiku.transform_with_state). 
154      If the model has no non-trainable state, it should be transformed with 
155      [`haiku.transform`]( 
156        https://dm-haiku.readthedocs.io/en/latest/api.html#haiku.transform). 
157      Additionally, and optionally, if the module does not use RNGs in "apply", it 
158      can be transformed with 
159      [`haiku.without_apply_rng`]( 
160        https://dm-haiku.readthedocs.io/en/latest/api.html#without-apply-rng). 
161   
162      The following example shows how to create a `JaxLayer` from a Haiku module 
163      that uses random number generators via `hk.next_rng_key()` and takes a 
164      training positional argument: 
165   
166      ```python 
167      class MyHaikuModule(hk.Module): 
168          def __call__(self, x, training): 
169              x = hk.Conv2D(32, (3, 3))(x) 
170              x = jax.nn.relu(x) 
171              x = hk.AvgPool((1, 2, 2, 1), (1, 2, 2, 1), "VALID")(x) 
172              x = hk.Flatten()(x) 
173              x = hk.Linear(200)(x) 
174              if training: 
175                  x = hk.dropout(rng=hk.next_rng_key(), rate=0.3, x=x) 
176              x = jax.nn.relu(x) 
177              x = hk.Linear(10)(x) 
178              x = jax.nn.softmax(x) 
179              return x 
180   
181      def my_haiku_module_fn(inputs, training): 
182          module = MyHaikuModule() 
183          return module(inputs, training) 
184   
185      transformed_module = hk.transform(my_haiku_module_fn) 
186   
187      keras_layer = JaxLayer( 
188          call_fn=transformed_module.apply, 
189          init_fn=transformed_module.init, 
190      ) 
191      ``` 
192   
193      Args: 
194          call_fn: The function to call the model. See description above for the 
195              list of arguments it takes and the outputs it returns. 
196          init_fn: the function to call to initialize the model. See description 
197              above for the list of arguments it takes and the outputs it returns. 
198              If `None`, then `params` and/or `state` must be provided. 
199        params: A `PyTree` containing all the model trainable parameters. This 
200              allows passing trained parameters or controlling the initialization. 
201              If both `params` and `state` are `None`, `init_fn` is called at 
202              build time to initialize the trainable parameters of the model. 
203        state: A `PyTree` containing all the model non-trainable state. This 
204              allows passing learned state or controlling the initialization. If 
205              both `params` and `state` are `None`, and `call_fn` takes a `state` 
206              argument, then `init_fn` is called at build time to initialize the 
207              non-trainable state of the model. 
208        seed: Seed for random number generator. Optional. 
209        dtype: The dtype of the layer's computations and weights. Can also be a 
210              `keras.DTypePolicy`. Optional. Defaults to the default policy. 
211      """
212  
213      def __init__(
214          self,
215          call_fn,
216          init_fn=None,
217          params=None,
218          state=None,
219          seed=None,
220          **kwargs,
221      ):
222          if backend.backend() != "jax":
223              raise ValueError(
224                  "JaxLayer is only supported with the JAX backend. Current "
225                  f"backend: {backend.backend()}"
226              )
227  
228          if init_fn is None and params is None and state is None:
229              raise ValueError(
230                  "`init_fn`, `params` and `state` cannot all be `None`."
231              )
232  
233          super().__init__(**kwargs)
234          self.call_fn = call_fn
235          self.init_fn = init_fn
236          self.seed_generator = backend.random.SeedGenerator(seed)
237          self.tracked_params = self._create_variables(params, trainable=True)
238          self.tracked_state = self._create_variables(state, trainable=False)
239          if self.params is not None or self.state is not None:
240              self.built = True
241  
242          self.call_fn_arguments = self._validate_signature(
243              call_fn,
244              "call_fn",
245              {"params", "state", "rng", "inputs", "training"},
246              {"inputs"},
247          )
248          self.has_state = "state" in self.call_fn_arguments
249  
250          if init_fn:
251              self.init_fn_arguments = self._validate_signature(
252                  init_fn, "init_fn", {"rng", "inputs", "training"}, {"inputs"}
253              )
254  
255      def _validate_signature(self, fn, fn_name, allowed, required):
256          fn_parameters = inspect.signature(fn).parameters
257          for parameter_name in required:
258              if parameter_name not in fn_parameters:
259                  raise ValueError(
260                      f"Missing required argument in `{fn_name}`: "
261                      f"`{parameter_name}`"
262                  )
263  
264          parameter_names = []
265          for parameter in fn_parameters.values():
266              if parameter.name not in allowed:
267                  raise ValueError(
268                      f"Unsupported argument in `{fn_name}`: `{parameter.name}`, "
269                      f"supported arguments are `{'`, `'.join(allowed)}`"
270                  )
271              parameter_names.append(parameter.name)
272  
273          return parameter_names
274  
275      @tracking.no_automatic_dependency_tracking
276      def _create_variables(self, values, trainable):
277          """Create a structure of variables from a structure of JAX arrays. 
278   
279          `values` is traversed via JAX's `tree_map`. When a leaf is a JAX array 
280          or a tensor-like object, a corresponding variable is created with it as 
281          the initial value. The resulting structure of variables is assigned to 
282          `self.params` or `self.state` depending on `trainable`. Then, a 
283          flattened version of the variables is returned for tracking. 
284          `self.params` or `self.state` are intentionally not tracked because 
285          structures like `TrackedList` interfere with `jax.tree_utils`. 
286          Note that leaf objects that are not JAX arrays and not tensor-like are 
287          left intact as they are assumed to be configuration used by the model. 
288   
289          Args: 
290              values: the structure of values to traverse. 
291              trainable: whether to create trainable variables. 
292   
293          Returns: 
294              flat list of variables initialized with `values` for tracking. 
295          """
296  
297          def create_variable(value):
298              if backend.is_tensor(value) or isinstance(
299                  value, (np.ndarray, np.generic)
300              ):
301                  dtype = value.dtype
302                  if is_float_dtype(dtype):
303                      dtype = None  # Use the layer dtype policy
304                  return self.add_weight(
305                      value.shape,
306                      initializer=value,
307                      dtype=dtype,
308                      trainable=trainable,
309                  )
310              elif isinstance(value, (bool, int, float)):
311                  dtype = standardize_dtype(type(value))
312                  if is_float_dtype(dtype):
313                      dtype = None  # Use the layer dtype policy
314                  return self.add_weight(
315                      (),
316                      initializer=backend.convert_to_tensor(value),
317                      dtype=dtype,
318                      trainable=trainable,
319                  )
320              else:
321                  return value
322  
323          # Use JAX's tree_map as it understands registered classes.
324          variables = jax.tree_util.tree_map(create_variable, values)
325  
326          if trainable:
327              self.params = variables
328          else:
329              self.state = variables
330  
331          flat_variables, _ = jax.tree_util.tree_flatten(variables)
332          return flat_variables
333  
334      def _get_init_rng(self):
335          """ 
336          Returns a JAX `PRNGKey` or structure of `PRNGKey`s to pass to `init_fn`. 
337   
338          By default, this returns a single `PRNGKey` retrieved by calling 
339          `self.seed_generator.next()`. Override this to return a different 
340          structure. 
341   
342          Returns: 
343              a JAX `PRNGKey` or structure of `PRNGKey`s that will be passed as 
344              the `rng` argument of `init_fn`. 
345          """
346          return self.seed_generator.next()
347  
348      def _get_call_rng(self, training):
349          """ 
350          Returns a JAX `PRNGKey` or structure of `PRNGKey`s to pass to `call_fn`. 
351   
352          By default, this returns a single `PRNGKey` retrieved by calling 
353          `self.seed_generator.next()` when `training` is `True`, and `None` when 
354          `training` is `False`. Override this to return a different structure or 
355          to pass RNGs in inference mode too. 
356   
357          Returns: 
358              a JAX `PRNGKey` or structure of `PRNGKey`s that will be passed as 
359              the `rng` argument of `call_fn`. 
360          """
361          if training:
362              return self.seed_generator.next()
363          else:
364              return None
365  
366      def build(self, input_shape):
367          if self.params is not None or self.state is not None:
368              return
369  
370          if jax_utils.is_in_jax_tracing_scope():
371              # This exception is not actually shown, it is caught and a detailed
372              # warning about calling 'build' is printed.
373              raise ValueError("'JaxLayer' cannot be built in tracing scope")
374  
375          # Initialize `params` and `state` if needed by calling `init_fn`.
376          def create_input(shape):
377              shape = [d if d is not None else 1 for d in shape]
378              return jax.numpy.ones(shape)
379  
380          init_inputs = tree.map_shape_structure(create_input, input_shape)
381          init_args = []
382          for argument_name in self.init_fn_arguments:
383              if argument_name == "rng":
384                  init_args.append(self._get_init_rng())
385              elif argument_name == "inputs":
386                  init_args.append(init_inputs)
387              elif argument_name == "training":
388                  init_args.append(True)
389  
390          init_result = self.init_fn(*init_args)
391          if self.has_state:
392              init_params, init_state = init_result
393          else:
394              init_params, init_state = init_result, None
395  
396          self.tracked_params = self._create_variables(
397              init_params, trainable=True
398          )
399          self.tracked_state = self._create_variables(init_state, trainable=False)
400          self.built = True
401  
402      def call(self, inputs, training=False):
403          def unwrap_variable(variable):
404              return None if variable is None else variable.value
405  
406          call_args = []
407          for argument_name in self.call_fn_arguments:
408              if argument_name == "params":
409                  call_args.append(
410                      jax.tree_util.tree_map(unwrap_variable, self.params)
411                  )
412              elif argument_name == "state":
413                  call_args.append(
414                      jax.tree_util.tree_map(unwrap_variable, self.state)
415                  )
416              elif argument_name == "rng":
417                  call_args.append(self._get_call_rng(training))
418              elif argument_name == "inputs":
419                  call_args.append(inputs)
420              elif argument_name == "training":
421                  call_args.append(training)
422  
423          def assign_state_to_variable(value, variable):
424              # This exists only to make debugging this error case easier.
425              if not hasattr(variable, "assign"):
426                  raise ValueError(
427                      "Structure mismatch: the structure of the state returned "
428                      "by `call` does not match the structure of the state at "
429                      "initialization time."
430                  )
431              variable.assign(value)
432  
433          if self.has_state:
434              predictions, new_state = self.call_fn(*call_args)
435              jax.tree_util.tree_map(
436                  assign_state_to_variable, new_state, self.state
437              )
438              return predictions
439          else:
440              return self.call_fn(*call_args)
441  
442      def get_config(self):
443          config = {
444              "call_fn": serialization_lib.serialize_keras_object(self.call_fn),
445              "init_fn": serialization_lib.serialize_keras_object(self.init_fn),
446          }
447          base_config = super().get_config()
448          return dict(list(base_config.items()) + list(config.items()))
449  
450      @classmethod
451      def from_config(cls, config):
452          call_fn = serialization_lib.deserialize_keras_object(config["call_fn"])
453          init_fn = serialization_lib.deserialize_keras_object(config["init_fn"])
454          config["call_fn"] = call_fn
455          config["init_fn"] = init_fn
456          return super().from_config(config)

459  @keras_export("keras.layers.FlaxLayer")
460  class FlaxLayer(JaxLayer):
461      """Keras Layer that wraps a [Flax](https://flax.readthedocs.io) module. 
462   
463      This layer enables the use of Flax components in the form of 
464      [`flax.linen.Module`]( 
465          https://flax.readthedocs.io/en/latest/api_reference/flax.linen/module.html) 
466      instances within Keras when using JAX as the backend for Keras. 
467   
468      The module method to use for the forward pass can be specified via the 
469      `method` argument and is `__call__` by default. This method must take the 
470      following arguments with these exact names: 
471   
472      - `self` if the method is bound to the module, which is the case for the 
473          default of `__call__`, and `module` otherwise to pass the module. 
474      - `inputs`: the inputs to the model, a JAX array or a `PyTree` of arrays. 
475      - `training` *(optional)*: an argument specifying if we're in training mode 
476          or inference mode, `True` is passed in training mode. 
477   
478      `FlaxLayer` handles the non-trainable state of your model and required RNGs 
479      automatically. Note that the `mutable` parameter of 
480      [`flax.linen.Module.apply()`]( 
481          https://flax.readthedocs.io/en/latest/api_reference/flax.linen/module.html#flax.linen.apply) 
482      is set to `DenyList(["params"])`, therefore making the assumption that all 
483      the variables outside of the "params" collection are non-trainable weights. 
484   
485      This example shows how to create a `FlaxLayer` from a Flax `Module` with 
486      the default `__call__` method and no training argument: 
487   
488      ```python 
489      class MyFlaxModule(flax.linen.Module): 
490          @flax.linen.compact 
491          def __call__(self, inputs): 
492              x = inputs 
493              x = flax.linen.Conv(features=32, kernel_size=(3, 3))(x) 
494              x = flax.linen.relu(x) 
495              x = flax.linen.avg_pool(x, window_shape=(2, 2), strides=(2, 2)) 
496              x = x.reshape((x.shape[0], -1))  # flatten 
497              x = flax.linen.Dense(features=200)(x) 
498              x = flax.linen.relu(x) 
499              x = flax.linen.Dense(features=10)(x) 
500              x = flax.linen.softmax(x) 
501              return x 
502   
503      flax_module = MyFlaxModule() 
504      keras_layer = FlaxLayer(flax_module) 
505      ``` 
506   
507      This example shows how to wrap the module method to conform to the required 
508      signature. This allows having multiple input arguments and a training 
509      argument that has a different name and values. This additionally shows how 
510      to use a function that is not bound to the module. 
511   
512      ```python 
513      class MyFlaxModule(flax.linen.Module): 
514          @flax.linen.compact 
515          def forward(self, input1, input2, deterministic): 
516              ... 
517              return outputs 
518   
519      def my_flax_module_wrapper(module, inputs, training): 
520          input1, input2 = inputs 
521          return module.forward(input1, input2, not training) 
522   
523      flax_module = MyFlaxModule() 
524      keras_layer = FlaxLayer( 
525          module=flax_module, 
526          method=my_flax_module_wrapper, 
527      ) 
528      ``` 
529   
530      Args: 
531          module: An instance of `flax.linen.Module` or subclass. 
532          method: The method to call the model. This is generally a method in the 
533              `Module`. If not provided, the `__call__` method is used. `method` 
534              can also be a function not defined in the `Module`, in which case it 
535              must take the `Module` as the first argument. It is used for both 
536              `Module.init` and `Module.apply`. Details are documented in the 
537              `method` argument of [`flax.linen.Module.apply()`]( 
538                https://flax.readthedocs.io/en/latest/api_reference/flax.linen/module.html#flax.linen.apply). 
539          variables: A `dict` containing all the variables of the module in the 
540              same format as what is returned by [`flax.linen.Module.init()`]( 
541                https://flax.readthedocs.io/en/latest/api_reference/flax.linen/module.html#flax.linen.init). 
542              It should contain a "params" key and, if applicable, other keys for 
543              collections of variables for non-trainable state. This allows 
544              passing trained parameters and learned non-trainable state or 
545              controlling the initialization. If `None` is passed, the module's 
546              `init` function is called at build time to initialize the variables 
547              of the model. 
548      """
549  
550      def __init__(
551          self,
552          module,
553          method=None,
554          variables=None,
555          **kwargs,
556      ):
557          # Late import to only require Flax when this is used.
558          from flax.core import scope as flax_scope
559  
560          if backend.backend() != "jax":
561              raise ValueError(
562                  "FlaxLayer is only supported with the JAX backend. Current "
563                  f"backend: {backend.backend()}"
564              )
565  
566          self.module = module
567          self.method = method
568  
569          apply_mutable = flax_scope.DenyList(["params"])
570  
571          def apply_with_training(params, state, rng, inputs, training):
572              return self.module.apply(
573                  self._params_and_state_to_variables(params, state),
574                  inputs,
575                  rngs=rng,
576                  method=self.method,
577                  mutable=apply_mutable,
578                  training=training,
579              )
580  
581          def apply_without_training(params, state, rng, inputs):
582              return self.module.apply(
583                  self._params_and_state_to_variables(params, state),
584                  inputs,
585                  rngs=rng,
586                  method=self.method,
587                  mutable=apply_mutable,
588              )
589  
590          def init_with_training(rng, inputs, training):
591              return self._variables_to_params_and_state(
592                  self.module.init(
593                      rng,
594                      inputs,
595                      method=self.method,
596                      training=training,
597                  )
598              )
599  
600          def init_without_training(rng, inputs):
601              return self._variables_to_params_and_state(
602                  self.module.init(
603                      rng,
604                      inputs,
605                      method=self.method,
606                  )
607              )
608  
609          if (
610              "training"
611              in inspect.signature(method or module.__call__).parameters
612          ):
613              call_fn, init_fn = apply_with_training, init_with_training
614          else:
615              call_fn, init_fn = apply_without_training, init_without_training
616  
617          params, state = self._variables_to_params_and_state(variables)
618  
619          super().__init__(
620              call_fn=call_fn,
621              init_fn=init_fn,
622              params=params,
623              state=state,
624              **kwargs,
625          )
626  
627      def _params_and_state_to_variables(self, params, state):
628          if params:
629              if state:
630                  return {**params, **state}
631              else:
632                  return params
633          elif state:
634              return state
635          return {}
636  
637      def _variables_to_params_and_state(self, variables):
638          # neither params nor state
639          if variables is None:
640              return None, None
641          # state only
642          if "params" not in variables:
643              return {}, variables
644          # params only
645          if len(variables) == 1:
646              return variables, {}
647          # both, we need to split
648          params = {"params": variables["params"]}
649          state = {k: v for k, v in variables.items() if k != "params"}
650          return params, state
651  
652      def _get_init_rng(self):
653          return {
654              "params": self.seed_generator.next(),
655              "dropout": self.seed_generator.next(),
656          }
657  
658      def _get_call_rng(self, training):
659          if training:
660              return {"dropout": self.seed_generator.next()}
661          else:
662              return {}

664      def get_config(self):
665          config_method = self.method
666          if (
667              hasattr(self.method, "__self__")
668              and self.method.__self__ == self.module
669          ):
670              # A method bound to the module is serialized by name.
671              config_method = self.method.__name__
672          config = {
673              "module": serialization_lib.serialize_keras_object(self.module),
674              "method": serialization_lib.serialize_keras_object(config_method),
675          }
676          base_config = super().get_config()
677          # call_fn and init_fn come from module, do not save them.
678          base_config.pop("call_fn")
679          base_config.pop("init_fn")
680          return dict(list(base_config.items()) + list(config.items()))
681  
682      @classmethod
683      def from_config(cls, config):
684          module = serialization_lib.deserialize_keras_object(config["module"])
685          method = serialization_lib.deserialize_keras_object(config["method"])
686          if isinstance(config["method"], str):
687              # Deserialize bound method from the module.
688              method = getattr(module, method)
689          config["module"] = module
690          config["method"] = method
691          return cls(**config)
