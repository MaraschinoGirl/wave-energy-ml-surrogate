26   class GaussianProcessRegressor(MultiOutputMixin, RegressorMixin, BaseEstimator):
27       """Gaussian process regression (GPR). 
28    
29       The implementation is based on Algorithm 2.1 of [RW2006]_. 
30    
31       In addition to standard scikit-learn estimator API, 
32       :class:`GaussianProcessRegressor`: 
33    
34       * allows prediction without prior fitting (based on the GP prior) 
35       * provides an additional method `sample_y(X)`, which evaluates samples 
36         drawn from the GPR (prior or posterior) at given inputs 
37       * exposes a method `log_marginal_likelihood(theta)`, which can be used 
38         externally for other ways of selecting hyperparameters, e.g., via 
39         Markov chain Monte Carlo. 
40    
41       To learn the difference between a point-estimate approach vs. a more 
42       Bayesian modelling approach, refer to the example entitled 
43       :ref:`sphx_glr_auto_examples_gaussian_process_plot_compare_gpr_krr.py`. 
44    
45       Read more in the :ref:`User Guide <gaussian_process>`. 
46    
47       .. versionadded:: 0.18 
48    
49       Parameters 
50       ---------- 
51       kernel : kernel instance, default=None 
52           The kernel specifying the covariance function of the GP. If None is 
53           passed, the kernel ``ConstantKernel(1.0, constant_value_bounds="fixed") 
54           * RBF(1.0, length_scale_bounds="fixed")`` is used as default. Note that 
55           the kernel hyperparameters are optimized during fitting unless the 
56           bounds are marked as "fixed". 
57    
58       alpha : float or ndarray of shape (n_samples,), default=1e-10 
59           Value added to the diagonal of the kernel matrix during fitting. 
60           This can prevent a potential numerical issue during fitting, by 
61           ensuring that the calculated values form a positive definite matrix. 
62           It can also be interpreted as the variance of additional Gaussian 
63           measurement noise on the training observations. Note that this is 
64           different from using a `WhiteKernel`. If an array is passed, it must 
65           have the same number of entries as the data used for fitting and is 
66           used as datapoint-dependent noise level. Allowing to specify the 
67           noise level directly as a parameter is mainly for convenience and 
68           for consistency with :class:`~sklearn.linear_model.Ridge`. 
69    
70       optimizer : "fmin_l_bfgs_b", callable or None, default="fmin_l_bfgs_b" 
71           Can either be one of the internally supported optimizers for optimizing 
72           the kernel's parameters, specified by a string, or an externally 
73           defined optimizer passed as a callable. If a callable is passed, it 
74           must have the signature:: 
75    
76               def optimizer(obj_func, initial_theta, bounds): 
77                   # * 'obj_func': the objective function to be minimized, which 
78                   #   takes the hyperparameters theta as a parameter and an 
79                   #   optional flag eval_gradient, which determines if the 
80                   #   gradient is returned additionally to the function value 
81                   # * 'initial_theta': the initial value for theta, which can be 
82                   #   used by local optimizers 
83                   # * 'bounds': the bounds on the values of theta 
84                   .... 
85                   # Returned are the best found hyperparameters theta and 
86                   # the corresponding value of the target function. 
87                   return theta_opt, func_min 
88    
89           Per default, the L-BFGS-B algorithm from `scipy.optimize.minimize` 
90           is used. If None is passed, the kernel's parameters are kept fixed. 
91           Available internal optimizers are: `{'fmin_l_bfgs_b'}`. 
92    
93       n_restarts_optimizer : int, default=0 
94           The number of restarts of the optimizer for finding the kernel's 
95           parameters which maximize the log-marginal likelihood. The first run 
96           of the optimizer is performed from the kernel's initial parameters, 
97           the remaining ones (if any) from thetas sampled log-uniform randomly 
98           from the space of allowed theta-values. If greater than 0, all bounds 
99           must be finite. Note that `n_restarts_optimizer == 0` implies that one 
100          run is performed. 
101   
102      normalize_y : bool, default=False 
103          Whether or not to normalize the target values `y` by removing the mean 
104          and scaling to unit-variance. This is recommended for cases where 
105          zero-mean, unit-variance priors are used. Note that, in this 
106          implementation, the normalisation is reversed before the GP predictions 
107          are reported. 
108   
109          .. versionchanged:: 0.23 
110   
111      copy_X_train : bool, default=True 
112          If True, a persistent copy of the training data is stored in the 
113          object. Otherwise, just a reference to the training data is stored, 
114          which might cause predictions to change if the data is modified 
115          externally. 
116   
117      n_targets : int, default=None 
118          The number of dimensions of the target values. Used to decide the number 
119          of outputs when sampling from the prior distributions (i.e. calling 
120          :meth:`sample_y` before :meth:`fit`). This parameter is ignored once 
121          :meth:`fit` has been called. 
122   
123          .. versionadded:: 1.3 
124   
125      random_state : int, RandomState instance or None, default=None 
126          Determines random number generation used to initialize the centers. 
127          Pass an int for reproducible results across multiple function calls. 
128          See :term:`Glossary <random_state>`. 
129   
130      Attributes 
131      ---------- 
132      X_train_ : array-like of shape (n_samples, n_features) or list of object 
133          Feature vectors or other representations of training data (also 
134          required for prediction). 
135   
136      y_train_ : array-like of shape (n_samples,) or (n_samples, n_targets) 
137          Target values in training data (also required for prediction). 
138   
139      kernel_ : kernel instance 
140          The kernel used for prediction. The structure of the kernel is the 
141          same as the one passed as parameter but with optimized hyperparameters. 
142   
143      L_ : array-like of shape (n_samples, n_samples) 
144          Lower-triangular Cholesky decomposition of the kernel in ``X_train_``. 
145   
146      alpha_ : array-like of shape (n_samples,) 
147          Dual coefficients of training data points in kernel space. 
148   
149      log_marginal_likelihood_value_ : float 
150          The log-marginal-likelihood of ``self.kernel_.theta``. 
151   
152      n_features_in_ : int 
153          Number of features seen during :term:`fit`. 
154   
155          .. versionadded:: 0.24 
156   
157      feature_names_in_ : ndarray of shape (`n_features_in_`,) 
158          Names of features seen during :term:`fit`. Defined only when `X` 
159          has feature names that are all strings. 
160   
161          .. versionadded:: 1.0 
162   
163      See Also 
164      -------- 
165      GaussianProcessClassifier : Gaussian process classification (GPC) 
166          based on Laplace approximation. 
167   
168      References 
169      ---------- 
170      .. [RW2006] `Carl E. Rasmussen and Christopher K.I. Williams, 
171         "Gaussian Processes for Machine Learning", 
172         MIT Press 2006 <https://www.gaussianprocess.org/gpml/chapters/RW.pdf>`_ 
173   
174      Examples 
175      -------- 
176      >>> from sklearn.datasets import make_friedman2 
177      >>> from sklearn.gaussian_process import GaussianProcessRegressor 
178      >>> from sklearn.gaussian_process.kernels import DotProduct, WhiteKernel 
179      >>> X, y = make_friedman2(n_samples=500, noise=0, random_state=0) 
180      >>> kernel = DotProduct() + WhiteKernel() 
181      >>> gpr = GaussianProcessRegressor(kernel=kernel, 
182      ...         random_state=0).fit(X, y) 
183      >>> gpr.score(X, y) 
184      0.3680... 
185      >>> gpr.predict(X[:2,:], return_std=True) 
186      (array([653.0..., 592.1...]), array([316.6..., 316.6...])) 
187      """
188

192          "optimizer": [StrOptions({"fmin_l_bfgs_b"}), callable, None],
193          "n_restarts_optimizer": [Interval(Integral, 0, None, closed="left")],
194          "normalize_y": ["boolean"],

200      def __init__(
201          self,
202          kernel=None,
203          *,
204          alpha=1e-10,
205          optimizer="fmin_l_bfgs_b",
206          n_restarts_optimizer=0,
207          normalize_y=False,
208          copy_X_train=True,
209          n_targets=None,
210          random_state=None,
211      ):
212          self.kernel = kernel
213          self.alpha = alpha
214          self.optimizer = optimizer
215          self.n_restarts_optimizer = n_restarts_optimizer
216          self.normalize_y = normalize_y
217          self.copy_X_train = copy_X_train
218          self.n_targets = n_targets
219          self.random_state = random_state

221      @_fit_context(prefer_skip_nested_validation=True)
222      def fit(self, X, y):
223          """Fit Gaussian process regression model. 
224   
225          Parameters 
226          ---------- 
227          X : array-like of shape (n_samples, n_features) or list of object 
228              Feature vectors or other representations of training data. 
229   
230          y : array-like of shape (n_samples,) or (n_samples, n_targets) 
231              Target values. 
232   
233          Returns 
234          ------- 
235          self : object 
236              GaussianProcessRegressor class instance. 
237          """
238          if self.kernel is None:  # Use an RBF kernel as default
239              self.kernel_ = C(1.0, constant_value_bounds="fixed") * RBF(
240                  1.0, length_scale_bounds="fixed"
241              )
242          else:
243              self.kernel_ = clone(self.kernel)
244  
245          self._rng = check_random_state(self.random_state)
246  
247          if self.kernel_.requires_vector_input:
248              dtype, ensure_2d = "numeric", True
249          else:
250              dtype, ensure_2d = None, False
251          X, y = validate_data(
252              self,
253              X,
254              y,
255              multi_output=True,
256              y_numeric=True,
257              ensure_2d=ensure_2d,
258              dtype=dtype,
259          )
260  
261          n_targets_seen = y.shape[1] if y.ndim > 1 else 1
262          if self.n_targets is not None and n_targets_seen != self.n_targets:
263              raise ValueError(
264                  "The number of targets seen in `y` is different from the parameter "
265                  f"`n_targets`. Got {n_targets_seen} != {self.n_targets}."
266              )
267  
268          # Normalize target value
269          if self.normalize_y:
270              self._y_train_mean = np.mean(y, axis=0)
271              self._y_train_std = _handle_zeros_in_scale(np.std(y, axis=0), copy=False)
272  
273              # Remove mean and make unit variance
274              y = (y - self._y_train_mean) / self._y_train_std
275  
276          else:
277              shape_y_stats = (y.shape[1],) if y.ndim == 2 else 1
278              self._y_train_mean = np.zeros(shape=shape_y_stats)
279              self._y_train_std = np.ones(shape=shape_y_stats)
280  
281          if np.iterable(self.alpha) and self.alpha.shape[0] != y.shape[0]:
282              if self.alpha.shape[0] == 1:
283                  self.alpha = self.alpha[0]
284              else:
285                  raise ValueError(
286                      "alpha must be a scalar or an array with same number of "
287                      f"entries as y. ({self.alpha.shape[0]} != {y.shape[0]})"
288                  )
289  
290          self.X_train_ = np.copy(X) if self.copy_X_train else X
291          self.y_train_ = np.copy(y) if self.copy_X_train else y
292  
293          if self.optimizer is not None and self.kernel_.n_dims > 0:
294              # Choose hyperparameters based on maximizing the log-marginal
295              # likelihood (potentially starting from several initial values)
296              def obj_func(theta, eval_gradient=True):
297                  if eval_gradient:
298                      lml, grad = self.log_marginal_likelihood(
299                          theta, eval_gradient=True, clone_kernel=False
300                      )
301                      return -lml, -grad
302                  else:
303                      return -self.log_marginal_likelihood(theta, clone_kernel=False)
304  
305              # First optimize starting from theta specified in kernel
306              optima = [
307                  (
308                      self._constrained_optimization(
309                          obj_func, self.kernel_.theta, self.kernel_.bounds
310                      )
311                  )
312              ]
313  
314              # Additional runs are performed from log-uniform chosen initial
315              # theta
316              if self.n_restarts_optimizer > 0:
317                  if not np.isfinite(self.kernel_.bounds).all():
318                      raise ValueError(
319                          "Multiple optimizer restarts (n_restarts_optimizer>0) "
320                          "requires that all bounds are finite."
321                      )
322                  bounds = self.kernel_.bounds
323                  for iteration in range(self.n_restarts_optimizer):
324                      theta_initial = self._rng.uniform(bounds[:, 0], bounds[:, 1])
325                      optima.append(
326                          self._constrained_optimization(obj_func, theta_initial, bounds)
327                      )
328              # Select result from run with minimal (negative) log-marginal
329              # likelihood
330              lml_values = list(map(itemgetter(1), optima))
331              self.kernel_.theta = optima[np.argmin(lml_values)][0]
332              self.kernel_._check_bounds_params()
333  
334              self.log_marginal_likelihood_value_ = -np.min(lml_values)
335          else:
336              self.log_marginal_likelihood_value_ = self.log_marginal_likelihood(
337                  self.kernel_.theta, clone_kernel=False
338              )
339  
340          # Precompute quantities required for predictions which are independent
341          # of actual query points
342          # Alg. 2.1, page 19, line 2 -> L = cholesky(K + sigma^2 I)
343          K = self.kernel_(self.X_train_)
344          K[np.diag_indices_from(K)] += self.alpha
345          try:
346              self.L_ = cholesky(K, lower=GPR_CHOLESKY_LOWER, check_finite=False)
347          except np.linalg.LinAlgError as exc:
348              exc.args = (
349                  (
350                      f"The kernel, {self.kernel_}, is not returning a positive "
351                      "definite matrix. Try gradually increasing the 'alpha' "
352                      "parameter of your GaussianProcessRegressor estimator."
353                  ),
354              ) + exc.args
355              raise
356          # Alg 2.1, page 19, line 3 -> alpha = L^T \ (L \ y)
357          self.alpha_ = cho_solve(
358              (self.L_, GPR_CHOLESKY_LOWER),
359              self.y_train_,
360              check_finite=False,
361          )
362          return self
363  
364      def predict(self, X, return_std=False, return_cov=False):
365          """Predict using the Gaussian process regression model. 
366   
367          We can also predict based on an unfitted model by using the GP prior. 
368          In addition to the mean of the predictive distribution, optionally also 
369          returns its standard deviation (`return_std=True`) or covariance 
370          (`return_cov=True`). Note that at most one of the two can be requested. 
371   
372          Parameters 
373          ---------- 
374          X : array-like of shape (n_samples, n_features) or list of object 
375              Query points where the GP is evaluated. 
376   
377          return_std : bool, default=False 
378              If True, the standard-deviation of the predictive distribution at 
379              the query points is returned along with the mean. 
380   
381          return_cov : bool, default=False 
382              If True, the covariance of the joint predictive distribution at 
383              the query points is returned along with the mean. 
384   
385          Returns 
386          ------- 
387          y_mean : ndarray of shape (n_samples,) or (n_samples, n_targets) 
388              Mean of predictive distribution at query points. 
389   
390          y_std : ndarray of shape (n_samples,) or (n_samples, n_targets), optional 
391              Standard deviation of predictive distribution at query points. 
392              Only returned when `return_std` is True. 
393   
394          y_cov : ndarray of shape (n_samples, n_samples) or \ 
395                  (n_samples, n_samples, n_targets), optional 
396              Covariance of joint predictive distribution at query points. 
397              Only returned when `return_cov` is True. 
398          """
399          if return_std and return_cov:
400              raise RuntimeError(
401                  "At most one of return_std or return_cov can be requested."
402              )
403  
404          if self.kernel is None or self.kernel.requires_vector_input:
405              dtype, ensure_2d = "numeric", True
406          else:
407              dtype, ensure_2d = None, False
408  
409          X = validate_data(self, X, ensure_2d=ensure_2d, dtype=dtype, reset=False)
410  
411          if not hasattr(self, "X_train_"):  # Unfitted;predict based on GP prior
412              if self.kernel is None:
413                  kernel = C(1.0, constant_value_bounds="fixed") * RBF(
414                      1.0, length_scale_bounds="fixed"
415                  )
416              else:
417                  kernel = self.kernel
418  
419              n_targets = self.n_targets if self.n_targets is not None else 1
420              y_mean = np.zeros(shape=(X.shape[0], n_targets)).squeeze()
421  
422              if return_cov:
423                  y_cov = kernel(X)
424                  if n_targets > 1:
425                      y_cov = np.repeat(
426                          np.expand_dims(y_cov, -1), repeats=n_targets, axis=-1
427                      )
428                  return y_mean, y_cov
429              elif return_std:
430                  y_var = kernel.diag(X)
431                  if n_targets > 1:
432                      y_var = np.repeat(
433                          np.expand_dims(y_var, -1), repeats=n_targets, axis=-1
434                      )
435                  return y_mean, np.sqrt(y_var)
436              else:
437                  return y_mean
438          else:  # Predict based on GP posterior
439              # Alg 2.1, page 19, line 4 -> f*_bar = K(X_test, X_train) . alpha
440              K_trans = self.kernel_(X, self.X_train_)
441              y_mean = K_trans @ self.alpha_
442  
443              # undo normalisation
444              y_mean = self._y_train_std * y_mean + self._y_train_mean
445  
446              # if y_mean has shape (n_samples, 1), reshape to (n_samples,)
447              if y_mean.ndim > 1 and y_mean.shape[1] == 1:
448                  y_mean = np.squeeze(y_mean, axis=1)
449  
450              # Alg 2.1, page 19, line 5 -> v = L \ K(X_test, X_train)^T
451              V = solve_triangular(
452                  self.L_, K_trans.T, lower=GPR_CHOLESKY_LOWER, check_finite=False
453              )
454  
455              if return_cov:
456                  # Alg 2.1, page 19, line 6 -> K(X_test, X_test) - v^T. v
457                  y_cov = self.kernel_(X) - V.T @ V
458  
459                  # undo normalisation
460                  y_cov = np.outer(y_cov, self._y_train_std**2).reshape(*y_cov.shape, -1)
461                  # if y_cov has shape (n_samples, n_samples, 1), reshape to
462                  # (n_samples, n_samples)
463                  if y_cov.shape[2] == 1:
464                      y_cov = np.squeeze(y_cov, axis=2)
465  
466                  return y_mean, y_cov
467              elif return_std:
468                  # Compute variance of predictive distribution
469                  # Use einsum to avoid explicitly forming the large matrix
470                  # V^T @ V just to extract its diagonal afterward.
471                  y_var = self.kernel_.diag(X).copy()
472                  y_var -= np.einsum("ij,ji->i", V.T, V)
473  
474                  # Check if any of the variances is negative because of
475                  # numerical issues. If yes: set the variance to 0.
476                  y_var_negative = y_var < 0
477                  if np.any(y_var_negative):
478                      warnings.warn(
479                          "Predicted variances smaller than 0. "
480                          "Setting those variances to 0."
481                      )
482                      y_var[y_var_negative] = 0.0
483  
484                  # undo normalisation
485                  y_var = np.outer(y_var, self._y_train_std**2).reshape(*y_var.shape, -1)
486  
487                  # if y_var has shape (n_samples, 1), reshape to (n_samples,)
488                  if y_var.shape[1] == 1:
489                      y_var = np.squeeze(y_var, axis=1)
490  
491                  return y_mean, np.sqrt(y_var)
492              else:
493                  return y_mean
494  
495      def sample_y(self, X, n_samples=1, random_state=0):
496          """Draw samples from Gaussian process and evaluate at X. 
497   
498          Parameters 
499          ---------- 
500          X : array-like of shape (n_samples_X, n_features) or list of object 
501              Query points where the GP is evaluated. 
502   
503          n_samples : int, default=1 
504              Number of samples drawn from the Gaussian process per query point. 
505   
506          random_state : int, RandomState instance or None, default=0 
507              Determines random number generation to randomly draw samples. 
508              Pass an int for reproducible results across multiple function 
509              calls. 
510              See :term:`Glossary <random_state>`. 
511   
512          Returns 
513          ------- 
514          y_samples : ndarray of shape (n_samples_X, n_samples), or \ 
515              (n_samples_X, n_targets, n_samples) 
516              Values of n_samples samples drawn from Gaussian process and 
517              evaluated at query points. 
518          """
519          rng = check_random_state(random_state)
520  
521          y_mean, y_cov = self.predict(X, return_cov=True)
522          if y_mean.ndim == 1:
523              y_samples = rng.multivariate_normal(y_mean, y_cov, n_samples).T
524          else:
525              y_samples = [
526                  rng.multivariate_normal(
527                      y_mean[:, target], y_cov[..., target], n_samples
528                  ).T[:, np.newaxis]
529                  for target in range(y_mean.shape[1])
530              ]
531              y_samples = np.hstack(y_samples)
532          return y_samples
533  
534      def log_marginal_likelihood(
535          self, theta=None, eval_gradient=False, clone_kernel=True
536      ):
537          """Return log-marginal likelihood of theta for training data. 
538   
539          Parameters 
540          ---------- 
541          theta : array-like of shape (n_kernel_params,) default=None 
542              Kernel hyperparameters for which the log-marginal likelihood is 
543              evaluated. If None, the precomputed log_marginal_likelihood 
544              of ``self.kernel_.theta`` is returned. 
545   
546          eval_gradient : bool, default=False 
547              If True, the gradient of the log-marginal likelihood with respect 
548              to the kernel hyperparameters at position theta is returned 
549              additionally. If True, theta must not be None. 
550   
551          clone_kernel : bool, default=True 
552              If True, the kernel attribute is copied. If False, the kernel 
553              attribute is modified, but may result in a performance improvement. 
554   
555          Returns 
556          ------- 
557          log_likelihood : float 
558              Log-marginal likelihood of theta for training data. 
559   
560          log_likelihood_gradient : ndarray of shape (n_kernel_params,), optional 
561              Gradient of the log-marginal likelihood with respect to the kernel 
562              hyperparameters at position theta. 
563              Only returned when eval_gradient is True. 
564          """
565          if theta is None:
566              if eval_gradient:
567                  raise ValueError("Gradient can only be evaluated for theta!=None")
568              return self.log_marginal_likelihood_value_
569  
570          if clone_kernel:
571              kernel = self.kernel_.clone_with_theta(theta)
572          else:
573              kernel = self.kernel_
574              kernel.theta = theta
575  
576          if eval_gradient:
577              K, K_gradient = kernel(self.X_train_, eval_gradient=True)
578          else:
579              K = kernel(self.X_train_)
580  
581          # Alg. 2.1, page 19, line 2 -> L = cholesky(K + sigma^2 I)
582          K[np.diag_indices_from(K)] += self.alpha
583          try:
584              L = cholesky(K, lower=GPR_CHOLESKY_LOWER, check_finite=False)
585          except np.linalg.LinAlgError:
586              return (-np.inf, np.zeros_like(theta)) if eval_gradient else -np.inf
587  
588          # Support multi-dimensional output of self.y_train_
589          y_train = self.y_train_
590          if y_train.ndim == 1:
591              y_train = y_train[:, np.newaxis]
592  
593          # Alg 2.1, page 19, line 3 -> alpha = L^T \ (L \ y)
594          alpha = cho_solve((L, GPR_CHOLESKY_LOWER), y_train, check_finite=False)
595  
596          # Alg 2.1, page 19, line 7
597          # -0.5 . y^T . alpha - sum(log(diag(L))) - n_samples / 2 log(2*pi)
598          # y is originally thought to be a (1, n_samples) row vector. However,
599          # in multioutputs, y is of shape (n_samples, 2) and we need to compute
600          # y^T . alpha for each output, independently using einsum. Thus, it
601          # is equivalent to:
602          # for output_idx in range(n_outputs):
603          #     log_likelihood_dims[output_idx] = (
604          #         y_train[:, [output_idx]] @ alpha[:, [output_idx]]
605          #     )
606          log_likelihood_dims = -0.5 * np.einsum("ik,ik->k", y_train, alpha)
607          log_likelihood_dims -= np.log(np.diag(L)).sum()
608          log_likelihood_dims -= K.shape[0] / 2 * np.log(2 * np.pi)
609          # the log likehood is sum-up across the outputs
610          log_likelihood = log_likelihood_dims.sum(axis=-1)
611  
612          if eval_gradient:
613              # Eq. 5.9, p. 114, and footnote 5 in p. 114
614              # 0.5 * trace((alpha . alpha^T - K^-1) . K_gradient)
615              # alpha is supposed to be a vector of (n_samples,) elements. With
616              # multioutputs, alpha is a matrix of size (n_samples, n_outputs).
617              # Therefore, we want to construct a matrix of
618              # (n_samples, n_samples, n_outputs) equivalent to
619              # for output_idx in range(n_outputs):
620              #     output_alpha = alpha[:, [output_idx]]
621              #     inner_term[..., output_idx] = output_alpha @ output_alpha.T
622              inner_term = np.einsum("ik,jk->ijk", alpha, alpha)
623              # compute K^-1 of shape (n_samples, n_samples)
624              K_inv = cho_solve(
625                  (L, GPR_CHOLESKY_LOWER), np.eye(K.shape[0]), check_finite=False
626              )
627              # create a new axis to use broadcasting between inner_term and
628              # K_inv
629              inner_term -= K_inv[..., np.newaxis]
630              # Since we are interested about the trace of
631              # inner_term @ K_gradient, we don't explicitly compute the
632              # matrix-by-matrix operation and instead use an einsum. Therefore
633              # it is equivalent to:
634              # for param_idx in range(n_kernel_params):
635              #     for output_idx in range(n_output):
636              #         log_likehood_gradient_dims[param_idx, output_idx] = (
637              #             inner_term[..., output_idx] @
638              #             K_gradient[..., param_idx]
639              #         )
640              log_likelihood_gradient_dims = 0.5 * np.einsum(
641                  "ijl,jik->kl", inner_term, K_gradient
642              )
643              # the log likehood gradient is the sum-up across the outputs
644              log_likelihood_gradient = log_likelihood_gradient_dims.sum(axis=-1)
645  
646          if eval_gradient:
647              return log_likelihood, log_likelihood_gradient
648          else:
649              return log_likelihood

651      def _constrained_optimization(self, obj_func, initial_theta, bounds):
652          if self.optimizer == "fmin_l_bfgs_b":
653              opt_res = scipy.optimize.minimize(
654                  obj_func,
655                  initial_theta,
656                  method="L-BFGS-B",
657                  jac=True,
658                  bounds=bounds,
659              )
660              _check_optimize_result("lbfgs", opt_res)
661              theta_opt, func_min = opt_res.x, opt_res.fun
662          elif callable(self.optimizer):
663              theta_opt, func_min = self.optimizer(obj_func, initial_theta, bounds=bounds)
664          else:
665              raise ValueError(f"Unknown optimizer {self.optimizer}.")
666  
667          return theta_opt, func_min
