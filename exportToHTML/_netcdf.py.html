_netcdf.py 
1    """ 
2    NetCDF reader/writer module. 
3     
4    This module is used to read and create NetCDF files. NetCDF files are 
5    accessed through the `netcdf_file` object. Data written to and from NetCDF 
6    files are contained in `netcdf_variable` objects. Attributes are given 
7    as member variables of the `netcdf_file` and `netcdf_variable` objects. 
8     
9    This module implements the Scientific.IO.NetCDF API to read and create 
10   NetCDF files. The same API is also used in the PyNIO and pynetcdf 
11   modules, allowing these modules to be used interchangeably when working 
12   with NetCDF files. 
13    
14   Only NetCDF3 is supported here; for NetCDF4 see 
15   `netCDF4-python <http://unidata.github.io/netcdf4-python/>`__, 
16   which has a similar API. 
17    
18   """

98   class netcdf_file:
99       """ 
100      A file object for NetCDF data. 
101   
102      A `netcdf_file` object has two standard attributes: `dimensions` and 
103      `variables`. The values of both are dictionaries, mapping dimension 
104      names to their associated lengths and variable names to variables, 
105      respectively. Application programs should never modify these 
106      dictionaries. 
107   
108      All other attributes correspond to global attributes defined in the 
109      NetCDF file. Global file attributes are created by assigning to an 
110      attribute of the `netcdf_file` object. 
111   
112      Parameters 
113      ---------- 
114      filename : string or file-like 
115          string -> filename 
116      mode : {'r', 'w', 'a'}, optional 
117          read-write-append mode, default is 'r' 
118      mmap : None or bool, optional 
119          Whether to mmap `filename` when reading.  Default is True 
120          when `filename` is a file name, False when `filename` is a 
121          file-like object. Note that when mmap is in use, data arrays 
122          returned refer directly to the mmapped data on disk, and the 
123          file cannot be closed as long as references to it exist. 
124      version : {1, 2}, optional 
125          version of netcdf to read / write, where 1 means *Classic 
126          format* and 2 means *64-bit offset format*.  Default is 1.  See 
127          `here <https://docs.unidata.ucar.edu/nug/current/netcdf_introduction.html#select_format>`__ 
128          for more info. 
129      maskandscale : bool, optional 
130          Whether to automatically scale and/or mask data based on attributes. 
131          Default is False. 
132   
133      Notes 
134      ----- 
135      The major advantage of this module over other modules is that it doesn't 
136      require the code to be linked to the NetCDF libraries. This module is 
137      derived from `pupynere <https://bitbucket.org/robertodealmeida/pupynere/>`_. 
138   
139      NetCDF files are a self-describing binary data format. The file contains 
140      metadata that describes the dimensions and variables in the file. More 
141      details about NetCDF files can be found `here 
142      <https://www.unidata.ucar.edu/software/netcdf/guide_toc.html>`__. There 
143      are three main sections to a NetCDF data structure: 
144   
145      1. Dimensions 
146      2. Variables 
147      3. Attributes 
148   
149      The dimensions section records the name and length of each dimension used 
150      by the variables. The variables would then indicate which dimensions it 
151      uses and any attributes such as data units, along with containing the data 
152      values for the variable. It is good practice to include a 
153      variable that is the same name as a dimension to provide the values for 
154      that axes. Lastly, the attributes section would contain additional 
155      information such as the name of the file creator or the instrument used to 
156      collect the data. 
157   
158      When writing data to a NetCDF file, there is often the need to indicate the 
159      'record dimension'. A record dimension is the unbounded dimension for a 
160      variable. For example, a temperature variable may have dimensions of 
161      latitude, longitude and time. If one wants to add more temperature data to 
162      the NetCDF file as time progresses, then the temperature variable should 
163      have the time dimension flagged as the record dimension. 
164   
165      In addition, the NetCDF file header contains the position of the data in 
166      the file, so access can be done in an efficient manner without loading 
167      unnecessary data into memory. It uses the ``mmap`` module to create 
168      Numpy arrays mapped to the data on disk, for the same purpose. 
169   
170      Note that when `netcdf_file` is used to open a file with mmap=True 
171      (default for read-only), arrays returned by it refer to data 
172      directly on the disk. The file should not be closed, and cannot be cleanly 
173      closed when asked, if such arrays are alive. You may want to copy data arrays 
174      obtained from mmapped Netcdf file if they are to be processed after the file 
175      is closed, see the example below. 
176   
177      Examples 
178      -------- 
179      To create a NetCDF file: 
180   
181      >>> from scipy.io import netcdf_file 
182      >>> import numpy as np 
183      >>> f = netcdf_file('simple.nc', 'w') 
184      >>> f.history = 'Created for a test' 
185      >>> f.createDimension('time', 10) 
186      >>> time = f.createVariable('time', 'i', ('time',)) 
187      >>> time[:] = np.arange(10) 
188      >>> time.units = 'days since 2008-01-01' 
189      >>> f.close() 
190   
191      Note the assignment of ``arange(10)`` to ``time[:]``.  Exposing the slice 
192      of the time variable allows for the data to be set in the object, rather 
193      than letting ``arange(10)`` overwrite the ``time`` variable. 
194   
195      To read the NetCDF file we just created: 
196   
197      >>> from scipy.io import netcdf_file 
198      >>> f = netcdf_file('simple.nc', 'r') 
199      >>> print(f.history) 
200      b'Created for a test' 
201      >>> time = f.variables['time'] 
202      >>> print(time.units) 
203      b'days since 2008-01-01' 
204      >>> print(time.shape) 
205      (10,) 
206      >>> print(time[-1]) 
207      9 
208   
209      NetCDF files, when opened read-only, return arrays that refer 
210      directly to memory-mapped data on disk: 
211   
212      >>> data = time[:] 
213   
214      If the data is to be processed after the file is closed, it needs 
215      to be copied to main memory: 
216   
217      >>> data = time[:].copy() 
218      >>> del time 
219      >>> f.close() 
220      >>> data.mean() 
221      4.5 
222   
223      A NetCDF file can also be used as context manager: 
224   
225      >>> from scipy.io import netcdf_file 
226      >>> with netcdf_file('simple.nc', 'r') as f: 
227      ...     print(f.history) 
228      b'Created for a test' 
229   
230      """
231      def __init__(self, filename, mode='r', mmap=None, version=1,
232                   maskandscale=False):
233          """Initialize netcdf_file from fileobj (str or file-like)."""
234          if mode not in 'rwa':
235              raise ValueError("Mode must be either 'r', 'w' or 'a'.")
236  
237          if hasattr(filename, 'seek'):  # file-like
238              self.fp = filename
239              self.filename = 'None'
240              if mmap is None:
241                  mmap = False
242              elif mmap and not hasattr(filename, 'fileno'):
243                  raise ValueError('Cannot use file object for mmap')
244          else:  # maybe it's a string
245              self.filename = filename
246              omode = 'r+' if mode == 'a' else mode
247              self.fp = open(self.filename, f'{omode}b')
248              if mmap is None:
249                  # Mmapped files on PyPy cannot be usually closed
250                  # before the GC runs, so it's better to use mmap=False
251                  # as the default.
252                  mmap = (not IS_PYPY)
253  
254          if mode != 'r':
255              # Cannot read write-only files
256              mmap = False
257  
258          self.use_mmap = mmap
259          self.mode = mode
260          self.version_byte = version
261          self.maskandscale = maskandscale
262  
263          self.dimensions = {}
264          self.variables = {}
265  
266          self._dims = []
267          self._recs = 0
268          self._recsize = 0
269  
270          self._mm = None
271          self._mm_buf = None
272          if self.use_mmap:
273              self._mm = mm.mmap(self.fp.fileno(), 0, access=mm.ACCESS_READ)
274              self._mm_buf = np.frombuffer(self._mm, dtype=np.int8)
275  
276          self._attributes = {}
277  
278          if mode in 'ra':
279              self._read()
280  
281      def __setattr__(self, attr, value):
282          # Store user defined attributes in a separate dict,
283          # so we can save them to file later.
284          try:
285              self._attributes[attr] = value
286          except AttributeError:
287              pass
288          self.__dict__[attr] = value
289  
290      def close(self):
291          """Closes the NetCDF file."""
292          if hasattr(self, 'fp') and not self.fp.closed:
293              try:
294                  self.flush()
295              finally:
296                  self.variables = {}
297                  if self._mm_buf is not None:
298                      ref = weakref.ref(self._mm_buf)
299                      self._mm_buf = None
300                      if ref() is None:
301                          # self._mm_buf is gc'd, and we can close the mmap
302                          self._mm.close()
303                      else:
304                          # we cannot close self._mm, since self._mm_buf is
305                          # alive and there may still be arrays referring to it
306                          warnings.warn(
307                              "Cannot close a netcdf_file opened with mmap=True, when "
308                              "netcdf_variables or arrays referring to its data still "
309                              "exist. All data arrays obtained from such files refer "
310                              "directly to data on disk, and must be copied before the "
311                              "file can be cleanly closed. "
312                              "(See netcdf_file docstring for more information on mmap.)",
313                              category=RuntimeWarning, stacklevel=2,
314                          )
315                  self._mm = None
316                  self.fp.close()
317      __del__ = close
318  
319      def __enter__(self):
320          return self
321  
322      def __exit__(self, type, value, traceback):
323          self.close()

325      def createDimension(self, name, length):
326          """ 
327          Adds a dimension to the Dimension section of the NetCDF data structure. 
328   
329          Note that this function merely adds a new dimension that the variables can 
330          reference. The values for the dimension, if desired, should be added as 
331          a variable using `createVariable`, referring to this dimension. 
332   
333          Parameters 
334          ---------- 
335          name : str 
336              Name of the dimension (Eg, 'lat' or 'time'). 
337          length : int 
338              Length of the dimension. 
339   
340          See Also 
341          -------- 
342          createVariable 
343   
344          """
345          if length is None and self._dims:
346              raise ValueError("Only first dimension may be unlimited!")
347  
348          self.dimensions[name] = length
349          self._dims.append(name)
350  
351      def createVariable(self, name, type, dimensions):
352          """ 
353          Create an empty variable for the `netcdf_file` object, specifying its data 
354          type and the dimensions it uses. 
355   
356          Parameters 
357          ---------- 
358          name : str 
359              Name of the new variable. 
360          type : dtype or str 
361              Data type of the variable. 
362          dimensions : sequence of str 
363              List of the dimension names used by the variable, in the desired order. 
364   
365          Returns 
366          ------- 
367          variable : netcdf_variable 
368              The newly created ``netcdf_variable`` object. 
369              This object has also been added to the `netcdf_file` object as well. 
370   
371          See Also 
372          -------- 
373          createDimension 
374   
375          Notes 
376          ----- 
377          Any dimensions to be used by the variable should already exist in the 
378          NetCDF data structure or should be created by `createDimension` prior to 
379          creating the NetCDF variable. 
380   
381          """
382          shape = tuple([self.dimensions[dim] for dim in dimensions])
383          shape_ = tuple([dim or 0 for dim in shape])  # replace None with 0 for NumPy
384  
385          type = dtype(type)
386          typecode, size = type.char, type.itemsize
387          if (typecode, size) not in REVERSE:
388              raise ValueError(f"NetCDF 3 does not support type {type}")
389  
390          # convert to big endian always for NetCDF 3
391          data = empty(shape_, dtype=type.newbyteorder("B"))
392          self.variables[name] = netcdf_variable(
393                  data, typecode, size, shape, dimensions,
394                  maskandscale=self.maskandscale)
395          return self.variables[name]
396  
397      def flush(self):
398          """ 
399          Perform a sync-to-disk flush if the `netcdf_file` object is in write mode. 
400   
401          See Also 
402          -------- 
403          sync : Identical function 
404   
405          """
406          if hasattr(self, 'mode') and self.mode in 'wa':
407              self._write()
408      sync = flush

410      def _write(self):
411          self.fp.seek(0)
412          self.fp.write(b'CDF')
413          self.fp.write(array(self.version_byte, '>b').tobytes())
414  
415          # Write headers and data.
416          self._write_numrecs()
417          self._write_dim_array()
418          self._write_gatt_array()
419          self._write_var_array()
420  
421      def _write_numrecs(self):
422          # Get highest record count from all record variables.
423          for var in self.variables.values():
424              if var.isrec and len(var.data) > self._recs:
425                  self.__dict__['_recs'] = len(var.data)
426          self._pack_int(self._recs)
427  
428      def _write_dim_array(self):
429          if self.dimensions:
430              self.fp.write(NC_DIMENSION)
431              self._pack_int(len(self.dimensions))
432              for name in self._dims:
433                  self._pack_string(name)
434                  length = self.dimensions[name]
435                  self._pack_int(length or 0)  # replace None with 0 for record dimension
436          else:
437              self.fp.write(ABSENT)
438  
439      def _write_gatt_array(self):
440          self._write_att_array(self._attributes)
441  
442      def _write_att_array(self, attributes):
443          if attributes:
444              self.fp.write(NC_ATTRIBUTE)
445              self._pack_int(len(attributes))
446              for name, values in attributes.items():
447                  self._pack_string(name)
448                  self._write_att_values(values)
449          else:
450              self.fp.write(ABSENT)
451  
452      def _write_var_array(self):
453          if self.variables:
454              self.fp.write(NC_VARIABLE)
455              self._pack_int(len(self.variables))
456  
457              # Sort variable names non-recs first, then recs.
458              def sortkey(n):
459                  v = self.variables[n]
460                  if v.isrec:
461                      return (-1,)
462                  return v._shape
463              variables = sorted(self.variables, key=sortkey, reverse=True)
464  
465              # Set the metadata for all variables.
466              for name in variables:
467                  self._write_var_metadata(name)
468              # Now that we have the metadata, we know the vsize of
469              # each record variable, so we can calculate recsize.
470              self.__dict__['_recsize'] = sum([
471                      var._vsize for var in self.variables.values()
472                      if var.isrec])
473              # Set the data for all variables.
474              for name in variables:
475                  self._write_var_data(name)
476          else:
477              self.fp.write(ABSENT)
478  
479      def _write_var_metadata(self, name):
480          var = self.variables[name]
481  
482          self._pack_string(name)
483          self._pack_int(len(var.dimensions))
484          for dimname in var.dimensions:
485              dimid = self._dims.index(dimname)
486              self._pack_int(dimid)
487  
488          self._write_att_array(var._attributes)
489  
490          nc_type = REVERSE[var.typecode(), var.itemsize()]
491          self.fp.write(nc_type)
492  
493          if not var.isrec:
494              vsize = var.data.size * var.data.itemsize
495              vsize += -vsize % 4
496          else:  # record variable
497              try:
498                  vsize = var.data[0].size * var.data.itemsize
499              except IndexError:
500                  vsize = 0
501              rec_vars = len([v for v in self.variables.values()
502                              if v.isrec])
503              if rec_vars > 1:
504                  vsize += -vsize % 4
505          self.variables[name].__dict__['_vsize'] = vsize
506          self._pack_int(vsize)
507  
508          # Pack a bogus begin, and set the real value later.
509          self.variables[name].__dict__['_begin'] = self.fp.tell()
510          self._pack_begin(0)
511  
512      def _write_var_data(self, name):
513          var = self.variables[name]
514  
515          # Set begin in file header.
516          the_beguine = self.fp.tell()
517          self.fp.seek(var._begin)
518          self._pack_begin(the_beguine)
519          self.fp.seek(the_beguine)
520  
521          # Write data.
522          if not var.isrec:
523              self.fp.write(var.data.tobytes())
524              count = var.data.size * var.data.itemsize
525              self._write_var_padding(var, var._vsize - count)
526          else:  # record variable
527              # Handle rec vars with shape[0] < nrecs.
528              if self._recs > len(var.data):
529                  shape = (self._recs,) + var.data.shape[1:]
530                  # Resize in-place does not always work since
531                  # the array might not be single-segment
532                  try:
533                      var.data.resize(shape)
534                  except ValueError:
535                      dtype = var.data.dtype
536                      var.__dict__['data'] = np.resize(var.data, shape).astype(dtype)
537  
538              pos0 = pos = self.fp.tell()
539              for rec in var.data:
540                  # Apparently scalars cannot be converted to big endian. If we
541                  # try to convert a ``=i4`` scalar to, say, '>i4' the dtype
542                  # will remain as ``=i4``.
543                  if not rec.shape and (rec.dtype.byteorder == '<' or
544                          (rec.dtype.byteorder == '=' and LITTLE_ENDIAN)):
545                      rec = rec.byteswap()
546                  self.fp.write(rec.tobytes())
547                  # Padding
548                  count = rec.size * rec.itemsize
549                  self._write_var_padding(var, var._vsize - count)
550                  pos += self._recsize
551                  self.fp.seek(pos)
552              self.fp.seek(pos0 + var._vsize)
553  
554      def _write_var_padding(self, var, size):
555          encoded_fill_value = var._get_encoded_fill_value()
556          num_fills = size // len(encoded_fill_value)
557          self.fp.write(encoded_fill_value * num_fills)
558  
559      def _write_att_values(self, values):
560          if hasattr(values, 'dtype'):
561              nc_type = REVERSE[values.dtype.char, values.dtype.itemsize]
562          else:
563              types = [(int, NC_INT), (float, NC_FLOAT), (str, NC_CHAR)]
564  
565              # bytes index into scalars in py3k. Check for "string" types
566              if isinstance(values, (str, bytes)):
567                  sample = values
568              else:
569                  try:
570                      sample = values[0]  # subscriptable?
571                  except TypeError:
572                      sample = values     # scalar
573  
574              for class_, nc_type in types:
575                  if isinstance(sample, class_):
576                      break
577  
578          typecode, size = TYPEMAP[nc_type]
579          dtype_ = f'>{typecode}'
580          # asarray() dies with bytes and '>c' in py3k. Change to 'S'
581          dtype_ = 'S' if dtype_ == '>c' else dtype_
582  
583          values = asarray(values, dtype=dtype_)
584  
585          self.fp.write(nc_type)
586  
587          if values.dtype.char == 'S':
588              nelems = values.itemsize
589          else:
590              nelems = values.size
591          self._pack_int(nelems)
592  
593          if not values.shape and (values.dtype.byteorder == '<' or
594                  (values.dtype.byteorder == '=' and LITTLE_ENDIAN)):
595              values = values.byteswap()
596          self.fp.write(values.tobytes())
597          count = values.size * values.itemsize
598          self.fp.write(b'\x00' * (-count % 4))  # pad
599  
600      def _read(self):
601          # Check magic bytes and version
602          magic = self.fp.read(3)
603          if not magic == b'CDF':
604              raise TypeError(f"Error: {self.filename} is not a valid NetCDF 3 file")
605          self.__dict__['version_byte'] = frombuffer(self.fp.read(1), '>b')[0]
606  
607          # Read file headers and set data.
608          self._read_numrecs()
609          self._read_dim_array()
610          self._read_gatt_array()
611          self._read_var_array()
612  
613      def _read_numrecs(self):
614          self.__dict__['_recs'] = self._unpack_int()
615  
616      def _read_dim_array(self):
617          header = self.fp.read(4)
618          if header not in [ZERO, NC_DIMENSION]:
619              raise ValueError("Unexpected header.")
620          count = self._unpack_int()
621  
622          for dim in range(count):
623              name = self._unpack_string().decode('latin1')
624              length = self._unpack_int() or None  # None for record dimension
625              self.dimensions[name] = length
626              self._dims.append(name)  # preserve order
627  
628      def _read_gatt_array(self):
629          for k, v in self._read_att_array().items():
630              self.__setattr__(k, v)
631  
632      def _read_att_array(self):
633          header = self.fp.read(4)
634          if header not in [ZERO, NC_ATTRIBUTE]:
635              raise ValueError("Unexpected header.")
636          count = self._unpack_int()
637  
638          attributes = {}
639          for attr in range(count):
640              name = self._unpack_string().decode('latin1')
641              attributes[name] = self._read_att_values()
642          return attributes
643  
644      def _read_var_array(self):
645          header = self.fp.read(4)
646          if header not in [ZERO, NC_VARIABLE]:
647              raise ValueError("Unexpected header.")
648  
649          begin = 0
650          dtypes = {'names': [], 'formats': []}
651          rec_vars = []
652          count = self._unpack_int()
653          for var in range(count):
654              (name, dimensions, shape, attributes,
655               typecode, size, dtype_, begin_, vsize) = self._read_var()
656              # https://www.unidata.ucar.edu/software/netcdf/guide_toc.html
657              # Note that vsize is the product of the dimension lengths
658              # (omitting the record dimension) and the number of bytes
659              # per value (determined from the type), increased to the
660              # next multiple of 4, for each variable. If a record
661              # variable, this is the amount of space per record. The
662              # netCDF "record size" is calculated as the sum of the
663              # vsize's of all the record variables.
664              #
665              # The vsize field is actually redundant, because its value
666              # may be computed from other information in the header. The
667              # 32-bit vsize field is not large enough to contain the size
668              # of variables that require more than 2^32 - 4 bytes, so
669              # 2^32 - 1 is used in the vsize field for such variables.
670              if shape and shape[0] is None:  # record variable
671                  rec_vars.append(name)
672                  # The netCDF "record size" is calculated as the sum of
673                  # the vsize's of all the record variables.
674                  self.__dict__['_recsize'] += vsize
675                  if begin == 0:
676                      begin = begin_
677                  dtypes['names'].append(name)
678                  dtypes['formats'].append(str(shape[1:]) + dtype_)
679  
680                  # Handle padding with a virtual variable.
681                  if typecode in 'bch':
682                      actual_size = reduce(mul, (1,) + shape[1:]) * size
683                      padding = -actual_size % 4
684                      if padding:
685                          dtypes['names'].append('_padding_%d' % var)
686                          dtypes['formats'].append('(%d,)>b' % padding)
687  
688                  # Data will be set later.
689                  data = None
690              else:  # not a record variable
691                  # Calculate size to avoid problems with vsize (above)
692                  a_size = reduce(mul, shape, 1) * size
693                  if self.use_mmap:
694                      data = self._mm_buf[begin_:begin_+a_size].view(dtype=dtype_)
695                      data.shape = shape
696                  else:
697                      pos = self.fp.tell()
698                      self.fp.seek(begin_)
699                      data = frombuffer(self.fp.read(a_size), dtype=dtype_
700                                        ).copy()
701                      data.shape = shape
702                      self.fp.seek(pos)
703  
704              # Add variable.
705              self.variables[name] = netcdf_variable(
706                      data, typecode, size, shape, dimensions, attributes,
707                      maskandscale=self.maskandscale)
708  
709          if rec_vars:
710              # Remove padding when only one record variable.
711              if len(rec_vars) == 1:
712                  dtypes['names'] = dtypes['names'][:1]
713                  dtypes['formats'] = dtypes['formats'][:1]
714  
715              # Build rec array.
716              if self.use_mmap:
717                  buf = self._mm_buf[begin:begin+self._recs*self._recsize]
718                  rec_array = buf.view(dtype=dtypes)
719                  rec_array.shape = (self._recs,)
720              else:
721                  pos = self.fp.tell()
722                  self.fp.seek(begin)
723                  rec_array = frombuffer(self.fp.read(self._recs*self._recsize),
724                                         dtype=dtypes).copy()
725                  rec_array.shape = (self._recs,)
726                  self.fp.seek(pos)
727  
728              for var in rec_vars:
729                  self.variables[var].__dict__['data'] = rec_array[var]
730  
731      def _read_var(self):
732          name = self._unpack_string().decode('latin1')
733          dimensions = []
734          shape = []
735          dims = self._unpack_int()
736  
737          for i in range(dims):
738              dimid = self._unpack_int()
739              dimname = self._dims[dimid]
740              dimensions.append(dimname)
741              dim = self.dimensions[dimname]
742              shape.append(dim)
743          dimensions = tuple(dimensions)
744          shape = tuple(shape)
745  
746          attributes = self._read_att_array()
747          nc_type = self.fp.read(4)
748          vsize = self._unpack_int()
749          begin = [self._unpack_int, self._unpack_int64][self.version_byte-1]()
750  
751          typecode, size = TYPEMAP[nc_type]
752          dtype_ = f'>{typecode}'
753  
754          return name, dimensions, shape, attributes, typecode, size, dtype_, begin, vsize
755  
756      def _read_att_values(self):
757          nc_type = self.fp.read(4)
758          n = self._unpack_int()
759  
760          typecode, size = TYPEMAP[nc_type]
761  
762          count = n*size
763          values = self.fp.read(int(count))
764          self.fp.read(-count % 4)  # read padding
765  
766          if typecode != 'c':
767              values = frombuffer(values, dtype=f'>{typecode}').copy()
768              if values.shape == (1,):
769                  values = values[0]
770          else:
771              values = values.rstrip(b'\x00')
772          return values
773  
774      def _pack_begin(self, begin):
775          if self.version_byte == 1:
776              self._pack_int(begin)
777          elif self.version_byte == 2:
778              self._pack_int64(begin)
779  
780      def _pack_int(self, value):
781          self.fp.write(array(value, '>i').tobytes())
782      _pack_int32 = _pack_int
783  
784      def _unpack_int(self):
785          return int(frombuffer(self.fp.read(4), '>i')[0])
786      _unpack_int32 = _unpack_int
787  
788      def _pack_int64(self, value):
789          self.fp.write(array(value, '>q').tobytes())
790  
791      def _unpack_int64(self):
792          return frombuffer(self.fp.read(8), '>q')[0]
793  
794      def _pack_string(self, s):
795          count = len(s)
796          self._pack_int(count)
797          self.fp.write(s.encode('latin1'))
798          self.fp.write(b'\x00' * (-count % 4))  # pad
799  
800      def _unpack_string(self):
801          count = self._unpack_int()
802          s = self.fp.read(count).rstrip(b'\x00')
803          self.fp.read(-count % 4)  # read padding
804          return s

807  class netcdf_variable:
808      """ 
809      A data object for netcdf files. 
810   
811      `netcdf_variable` objects are constructed by calling the method 
812      `netcdf_file.createVariable` on the `netcdf_file` object. `netcdf_variable` 
813      objects behave much like array objects defined in numpy, except that their 
814      data resides in a file. Data is read by indexing and written by assigning 
815      to an indexed subset; the entire array can be accessed by the index ``[:]`` 
816      or (for scalars) by using the methods `getValue` and `assignValue`. 
817      `netcdf_variable` objects also have attribute `shape` with the same meaning 
818      as for arrays, but the shape cannot be modified. There is another read-only 
819      attribute `dimensions`, whose value is the tuple of dimension names. 
820   
821      All other attributes correspond to variable attributes defined in 
822      the NetCDF file. Variable attributes are created by assigning to an 
823      attribute of the `netcdf_variable` object. 
824   
825      Parameters 
826      ---------- 
827      data : array_like 
828          The data array that holds the values for the variable. 
829          Typically, this is initialized as empty, but with the proper shape. 
830      typecode : dtype character code 
831          Desired data-type for the data array. 
832      size : int 
833          Desired element size for the data array. 
834      shape : sequence of ints 
835          The shape of the array. This should match the lengths of the 
836          variable's dimensions. 
837      dimensions : sequence of strings 
838          The names of the dimensions used by the variable. Must be in the 
839          same order of the dimension lengths given by `shape`. 
840      attributes : dict, optional 
841          Attribute values (any type) keyed by string names. These attributes 
842          become attributes for the netcdf_variable object. 
843      maskandscale : bool, optional 
844          Whether to automatically scale and/or mask data based on attributes. 
845          Default is False. 
846   
847   
848      Attributes 
849      ---------- 
850      dimensions : list of str 
851          List of names of dimensions used by the variable object. 
852      isrec, shape 
853          Properties 
854   
855      See also 
856      -------- 
857      isrec, shape 
858   
859      """
860      def __init__(self, data, typecode, size, shape, dimensions,
861                   attributes=None,
862                   maskandscale=False):
863          self.data = data
864          self._typecode = typecode
865          self._size = size
866          self._shape = shape
867          self.dimensions = dimensions
868          self.maskandscale = maskandscale
869  
870          self._attributes = attributes or {}
871          for k, v in self._attributes.items():
872              self.__dict__[k] = v
873  
874      def __setattr__(self, attr, value):
875          # Store user defined attributes in a separate dict,
876          # so we can save them to file later.
877          try:
878              self._attributes[attr] = value
879          except AttributeError:
880              pass
881          self.__dict__[attr] = value
882  
883      def isrec(self):
884          """Returns whether the variable has a record dimension or not. 
885   
886          A record dimension is a dimension along which additional data could be 
887          easily appended in the netcdf data structure without much rewriting of 
888          the data file. This attribute is a read-only property of the 
889          `netcdf_variable`. 
890   
891          """
892          return bool(self.data.shape) and not self._shape[0]
893      isrec = property(isrec)
894  
895      def shape(self):
896          """Returns the shape tuple of the data variable. 
897   
898          This is a read-only attribute and can not be modified in the 
899          same manner of other numpy arrays. 
900          """
901          return self.data.shape
902      shape = property(shape)
903  
904      def getValue(self):
905          """ 
906          Retrieve a scalar value from a `netcdf_variable` of length one. 
907   
908          Raises 
909          ------ 
910          ValueError 
911              If the netcdf variable is an array of length greater than one, 
912              this exception will be raised. 
913   
914          """
915          return self.data.item()
916  
917      def assignValue(self, value):
918          """ 
919          Assign a scalar value to a `netcdf_variable` of length one. 
920   
921          Parameters 
922          ---------- 
923          value : scalar 
924              Scalar value (of compatible type) to assign to a length-one netcdf 
925              variable. This value will be written to file. 
926   
927          Raises 
928          ------ 
929          ValueError 
930              If the input is not a scalar, or if the destination is not a length-one 
931              netcdf variable. 
932   
933          """
934          if not self.data.flags.writeable:
935              # Work-around for a bug in NumPy.  Calling itemset() on a read-only
936              # memory-mapped array causes a seg. fault.
937              # See NumPy ticket #1622, and SciPy ticket #1202.
938              # This check for `writeable` can be removed when the oldest version
939              # of NumPy still supported by scipy contains the fix for #1622.
940              raise RuntimeError("variable is not writeable")
941  
942          self.data[:] = value
943  
944      def typecode(self):
945          """ 
946          Return the typecode of the variable. 
947   
948          Returns 
949          ------- 
950          typecode : char 
951              The character typecode of the variable (e.g., 'i' for int). 
952   
953          """
954          return self._typecode
955  
956      def itemsize(self):
957          """ 
958          Return the itemsize of the variable. 
959   
960          Returns 
961          ------- 
962          itemsize : int 
963              The element size of the variable (e.g., 8 for float64). 
964   
965          """
966          return self._size
967  
968      def __getitem__(self, index):
969          if not self.maskandscale:
970              return self.data[index]
971  
972          data = self.data[index].copy()
973          missing_value = self._get_missing_value()
974          data = self._apply_missing_value(data, missing_value)
975          scale_factor = self._attributes.get('scale_factor')
976          add_offset = self._attributes.get('add_offset')
977          if add_offset is not None or scale_factor is not None:
978              data = data.astype(np.float64)
979          if scale_factor is not None:
980              data = data * scale_factor
981          if add_offset is not None:
982              data += add_offset
983  
984          return data
985  
986      def __setitem__(self, index, data):
987          if self.maskandscale:
988              missing_value = (
989                      self._get_missing_value() or
990                      getattr(data, 'fill_value', 999999))
991              self._attributes.setdefault('missing_value', missing_value)
992              self._attributes.setdefault('_FillValue', missing_value)
993              data = ((data - self._attributes.get('add_offset', 0.0)) /
994                      self._attributes.get('scale_factor', 1.0))
995              data = np.ma.asarray(data).filled(missing_value)
996              if self._typecode not in 'fd' and data.dtype.kind == 'f':
997                  data = np.round(data)
998  
999          # Expand data for record vars?
1000         if self.isrec:
1001             if isinstance(index, tuple):
1002                 rec_index = index[0]
1003             else:
1004                 rec_index = index
1005             if isinstance(rec_index, slice):
1006                 recs = (rec_index.start or 0) + len(data)
1007             else:
1008                 recs = rec_index + 1
1009             if recs > len(self.data):
1010                 shape = (recs,) + self._shape[1:]
1011                 # Resize in-place does not always work since
1012                 # the array might not be single-segment
1013                 try:
1014                     self.data.resize(shape)
1015                 except ValueError:
1016                     dtype = self.data.dtype
1017                     self.__dict__['data'] = np.resize(self.data, shape).astype(dtype)
1018         self.data[index] = data
1019 
1020     def _default_encoded_fill_value(self):
1021         """ 
1022         The default encoded fill-value for this Variable's data type. 
1023         """
1024         nc_type = REVERSE[self.typecode(), self.itemsize()]
1025         return FILLMAP[nc_type]
1026 
1027     def _get_encoded_fill_value(self):
1028         """ 
1029         Returns the encoded fill value for this variable as bytes. 
1030  
1031         This is taken from either the _FillValue attribute, or the default fill 
1032         value for this variable's data type. 
1033         """
1034         if '_FillValue' in self._attributes:
1035             fill_value = np.array(self._attributes['_FillValue'],
1036                                   dtype=self.data.dtype).tobytes()
1037             if len(fill_value) == self.itemsize():
1038                 return fill_value
1039             else:
1040                 return self._default_encoded_fill_value()
1041         else:
1042             return self._default_encoded_fill_value()
1043 
1044     def _get_missing_value(self):
1045         """ 
1046         Returns the value denoting "no data" for this variable. 
1047  
1048         If this variable does not have a missing/fill value, returns None. 
1049  
1050         If both _FillValue and missing_value are given, give precedence to 
1051         _FillValue. The netCDF standard gives special meaning to _FillValue; 
1052         missing_value is  just used for compatibility with old datasets. 
1053         """
1054 
1055         if '_FillValue' in self._attributes:
1056             missing_value = self._attributes['_FillValue']
1057         elif 'missing_value' in self._attributes:
1058             missing_value = self._attributes['missing_value']
1059         else:
1060             missing_value = None
1061 
1062         return missing_value
1063 
1064     @staticmethod
1065     def _apply_missing_value(data, missing_value):
1066         """ 
1067         Applies the given missing value to the data array. 
1068  
1069         Returns a numpy.ma array, with any value equal to missing_value masked 
1070         out (unless missing_value is None, in which case the original array is 
1071         returned). 
1072         """
1073 
1074         if missing_value is None:
1075             newdata = data
1076         else:
1077             try:
1078                 missing_value_isnan = np.isnan(missing_value)
1079             except (TypeError, NotImplementedError):
1080                 # some data types (e.g., characters) cannot be tested for NaN
1081                 missing_value_isnan = False
1082 
1083             if missing_value_isnan:
1084                 mymask = np.isnan(data)
1085             else:
1086                 mymask = (data == missing_value)
1087 
1088             newdata = np.ma.masked_where(mymask, data)
1089 
1090         return newdata
