81   class gaussian_kde:
82       """Representation of a kernel-density estimate using Gaussian kernels. 
83    
84       Kernel density estimation is a way to estimate the probability density 
85       function (PDF) of a random variable in a non-parametric way. 
86       `gaussian_kde` works for both uni-variate and multi-variate data.   It 
87       includes automatic bandwidth determination.  The estimation works best for 
88       a unimodal distribution; bimodal or multi-modal distributions tend to be 
89       oversmoothed. 
90    
91       Parameters 
92       ---------- 
93       dataset : array_like 
94           Datapoints to estimate from. In case of univariate data this is a 1-D 
95           array, otherwise a 2-D array with shape (# of dims, # of data). 
96       bw_method : str, scalar or callable, optional 
97           The method used to calculate the estimator bandwidth.  This can be 
98           'scott', 'silverman', a scalar constant or a callable.  If a scalar, 
99           this will be used directly as `kde.factor`.  If a callable, it should 
100          take a `gaussian_kde` instance as only parameter and return a scalar. 
101          If None (default), 'scott' is used.  See Notes for more details. 
102      weights : array_like, optional 
103          weights of datapoints. This must be the same shape as dataset. 
104          If None (default), the samples are assumed to be equally weighted 
105   
106      Attributes 
107      ---------- 
108      dataset : ndarray 
109          The dataset with which `gaussian_kde` was initialized. 
110      d : int 
111          Number of dimensions. 
112      n : int 
113          Number of datapoints. 
114      neff : int 
115          Effective number of datapoints. 
116   
117          .. versionadded:: 1.2.0 
118      factor : float 
119          The bandwidth factor, obtained from `kde.covariance_factor`, with which 
120          the covariance matrix is multiplied. 
121      covariance : ndarray 
122          The covariance matrix of `dataset`, scaled by the calculated bandwidth 
123          (`kde.factor`). 
124      inv_cov : ndarray 
125          The inverse of `covariance`. 
126   
127      Methods 
128      ------- 
129      evaluate 
130      __call__ 
131      integrate_gaussian 
132      integrate_box_1d 
133      integrate_box 
134      integrate_kde 
135      pdf 
136      logpdf 
137      resample 
138      set_bandwidth 
139      covariance_factor 
140   
141      Notes 
142      ----- 
143      Bandwidth selection strongly influences the estimate obtained from the KDE 
144      (much more so than the actual shape of the kernel).  Bandwidth selection 
145      can be done by a "rule of thumb", by cross-validation, by "plug-in 
146      methods" or by other means; see [3]_, [4]_ for reviews.  `gaussian_kde` 
147      uses a rule of thumb, the default is Scott's Rule. 
148   
149      Scott's Rule [1]_, implemented as `scotts_factor`, is:: 
150   
151          n**(-1./(d+4)), 
152   
153      with ``n`` the number of data points and ``d`` the number of dimensions. 
154      In the case of unequally weighted points, `scotts_factor` becomes:: 
155   
156          neff**(-1./(d+4)), 
157   
158      with ``neff`` the effective number of datapoints. 
159      Silverman's Rule [2]_, implemented as `silverman_factor`, is:: 
160   
161          (n * (d + 2) / 4.)**(-1. / (d + 4)). 
162   
163      or in the case of unequally weighted points:: 
164   
165          (neff * (d + 2) / 4.)**(-1. / (d + 4)). 
166   
167      Good general descriptions of kernel density estimation can be found in [1]_ 
168      and [2]_, the mathematics for this multi-dimensional implementation can be 
169      found in [1]_. 
170   
171      With a set of weighted samples, the effective number of datapoints ``neff`` 
172      is defined by:: 
173   
174          neff = sum(weights)^2 / sum(weights^2) 
175   
176      as detailed in [5]_. 
177   
178      References 
179      ---------- 
180      .. [1] D.W. Scott, "Multivariate Density Estimation: Theory, Practice, and 
181             Visualization", John Wiley & Sons, New York, Chicester, 1992. 
182      .. [2] B.W. Silverman, "Density Estimation for Statistics and Data 
183             Analysis", Vol. 26, Monographs on Statistics and Applied Probability, 
184             Chapman and Hall, London, 1986. 
185      .. [3] B.A. Turlach, "Bandwidth Selection in Kernel Density Estimation: A 
186             Review", CORE and Institut de Statistique, Vol. 19, pp. 1-33, 1993. 
187      .. [4] D.M. Bashtannyk and R.J. Hyndman, "Bandwidth selection for kernel 
188             conditional density estimation", Computational Statistics & Data 
189             Analysis, Vol. 36, pp. 279-298, 2001. 
190      .. [5] Gray P. G., 1969, Journal of the Royal Statistical Society. 
191             Series A (General), 132, 272 
192   
193      """
194      def __init__(self, dataset, bw_method=None, weights=None):
195          self.dataset = atleast_2d(asarray(dataset))
196          if not self.dataset.size > 1:
197              raise ValueError("`dataset` input should have multiple elements.")
198  
199          self.d, self.n = self.dataset.shape
200  
201          if weights is not None:
202              self._weights = atleast_1d(weights).astype(float)
203              self._weights /= sum(self._weights)
204              if self.weights.ndim != 1:
205                  raise ValueError("`weights` input should be one-dimensional.")
206              if len(self._weights) != self.n:
207                  raise ValueError("`weights` input should be of length n")
208              self._neff = 1/sum(self._weights**2)
209  
210          self.set_bandwidth(bw_method=bw_method)
211  
212      def evaluate(self, points):
213          """Evaluate the estimated pdf on a set of points. 
214   
215          Parameters 
216          ---------- 
217          points : (# of dimensions, # of points)-array 
218              Alternatively, a (# of dimensions,) vector can be passed in and 
219              treated as a single point. 
220   
221          Returns 
222          ------- 
223          values : (# of points,)-array 
224              The values at each point. 
225   
226          Raises 
227          ------ 
228          ValueError : if the dimensionality of the input points is different than 
229                       the dimensionality of the KDE. 
230   
231          """
232          points = atleast_2d(asarray(points))
233  
234          d, m = points.shape
235          if d != self.d:
236              if d == 1 and m == self.d:
237                  # points was passed in as a row vector
238                  points = reshape(points, (self.d, 1))
239                  m = 1
240              else:
241                  msg = f"points have dimension {d}, dataset has dimension {self.d}"
242                  raise ValueError(msg)
243  
244          output_dtype = np.common_type(self.covariance, points)
245          result = zeros((m,), dtype=output_dtype)
246  
247          whitening = linalg.cholesky(self.inv_cov)
248          scaled_dataset = dot(whitening, self.dataset)
249          scaled_points = dot(whitening, points)
250  
251          if m >= self.n:
252              # there are more points than data, so loop over data
253              for i in range(self.n):
254                  diff = scaled_dataset[:, i, newaxis] - scaled_points
255                  energy = sum(diff * diff, axis=0) / 2.0
256                  result += self.weights[i]*exp(-energy)
257          else:
258              # loop over points
259              for i in range(m):
260                  diff = scaled_dataset - scaled_points[:, i, newaxis]
261                  energy = sum(diff * diff, axis=0) / 2.0
262                  result[i] = sum(exp(-energy)*self.weights, axis=0)
263  
264          result = result / self._norm_factor
265  
266          return result
267  
268      __call__ = evaluate
269  
270      def scotts_factor(self):
271          """Compute Scott's factor. 
272   
273          Returns 
274          ------- 
275          s : float 
276              Scott's factor. 
277          """
278          return power(self.neff, -1./(self.d+4))
279  
280      def silverman_factor(self):
281          """Compute the Silverman factor. 
282   
283          Returns 
284          ------- 
285          s : float 
286              The silverman factor. 
287          """
288          return power(self.neff*(self.d+2.0)/4.0, -1./(self.d+4))
289  
290      #  Default method to calculate bandwidth, can be overwritten by subclass
291      covariance_factor = scotts_factor
292      covariance_factor.__doc__ = """Computes the coefficient (`kde.factor`) that 
293          multiplies the data covariance matrix to obtain the kernel covariance 
294          matrix. The default is `scotts_factor`.  A subclass can overwrite this 
295          method to provide a different method, or set it through a call to 
296          `kde.set_bandwidth`."""
297  
298      def set_bandwidth(self, bw_method=None):
299          """Compute the estimator bandwidth with given method. 
300   
301          The new bandwidth calculated after a call to `set_bandwidth` is used 
302          for subsequent evaluations of the estimated density. 
303   
304          Parameters 
305          ---------- 
306          bw_method : str, scalar or callable, optional 
307              The method used to calculate the estimator bandwidth.  This can be 
308              'scott', 'silverman', a scalar constant or a callable.  If a 
309              scalar, this will be used directly as `kde.factor`.  If a callable, 
310              it should take a `gaussian_kde` instance as only parameter and 
311              return a scalar.  If None (default), nothing happens; the current 
312              `kde.covariance_factor` method is kept. 
313   
314          Notes 
315          ----- 
316          .. versionadded:: 0.11 
317   
318          """
319          if bw_method is None:
320              pass
321          elif bw_method == 'scott':
322              self.covariance_factor = self.scotts_factor
323          elif bw_method == 'silverman':
324              self.covariance_factor = self.silverman_factor
325          elif np.isscalar(bw_method) and not isinstance(bw_method, str):
326              self._bw_method = 'use constant'
327              self.covariance_factor = lambda: bw_method
328          elif callable(bw_method):
329              self._bw_method = bw_method
330              self.covariance_factor = lambda: self._bw_method(self)
331          else:
332              msg = "`bw_method` should be 'scott', 'silverman', a scalar " \
333                    "or a callable."
334              raise ValueError(msg)
335  
336          self._compute_covariance()
337  
338      def _compute_covariance(self):
339          """Computes the covariance matrix for each Gaussian kernel using 
340          covariance_factor(). 
341          """
342          self.factor = self.covariance_factor()
343          # Cache covariance and inverse covariance of the data
344          if not hasattr(self, '_data_inv_cov'):
345              self._data_covariance = atleast_2d(cov(self.dataset, rowvar=1,
346                                                 bias=False,
347                                                 aweights=self.weights))
348              self._data_inv_cov = linalg.inv(self._data_covariance)
349  
350          self.covariance = self._data_covariance * self.factor**2
351          self.inv_cov = self._data_inv_cov / self.factor**2
352          self._norm_factor = sqrt(linalg.det(2*pi*self.covariance))
353  
354      def pdf(self, x):
355          """ 
356          Evaluate the estimated pdf on a provided set of points. 
357   
358          Notes 
359          ----- 
360          This is an alias for `gaussian_kde.evaluate`.  See the ``evaluate`` 
361          docstring for more details. 
362   
363          """
364          return self.evaluate(x)
365  
366      @property
367      def weights(self):
368          try:
369              return self._weights
370          except AttributeError:
371              self._weights = ones(self.n)/self.n
372              return self._weights
373  
374      @property
375      def neff(self):
376          try:
377              return self._neff
378          except AttributeError:
379              self._neff = 1/sum(self.weights**2)
380              return self._neff
